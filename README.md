# Astro: tabler icons plugin

**NOTE:** this only packages up and supports the tabler icons outline variant.

---

## Get started

Install the package:

```sh
npm i -D @cdransf/astro-tabler-icons
```

## Usage

```astro
---
import { IconCircleX } from "@cdransf/astro-tabler-icons";
---

<IconCircleX size={24} />

```
